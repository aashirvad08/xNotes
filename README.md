# xNotes 

A clean and fast mobile app built with React Native (Expo) for capturing notes, managing tasks, and organizing daily work in one place.

Designed for speed, simplicity, and structured thinking.

---

##  Features

- Create, edit, and delete notes
- Manage tasks alongside notes
- Organize content into folders
- Pin or prioritize important items
- Mark items as completed or active
- Smart priority suggestions
- Manual priority control
- Real-time updates
- Minimal dark UI

---

##  Smart Organization

xNotes helps structure your workflow:

- Prioritizes important items automatically  
- Keeps frequently used content accessible  
- Supports both quick notes and structured tasks
